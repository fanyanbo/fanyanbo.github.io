酷开网页开发技术文档

不知道从哪里开始？我们准备了一些你可以用来开始构建的便捷指南和参考文档。

开始出发

# 像数 1、2、3 一样的简单

# 安装 命令行工具
npm install -g ccos-cli

# 安装默认模版
ccos init .

# 安装包依赖
cd . && npm install

# 启动本地开发环境即可看到效果
npm run dev

TIP

ccos-cli 要求 Node.js 版本必须 >= 8.9.4。

# 简单易用的API

开箱即用，你需要做的只是在合适的业务中使用它。
