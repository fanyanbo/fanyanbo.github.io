## 像数 1、2、3 一样的简单

```bash
# 安装 命令行工具
npm install -g ccos-cli

# 安装默认模版
ccos init .

# 安装包依赖
cd . && npm install

# 启动本地开发环境即可看到效果
npm run dev
```

::: tip
ccos-cli 要求 [Node.js](http://nodejs.org/) 版本必须 >= 8.9.4。
:::
